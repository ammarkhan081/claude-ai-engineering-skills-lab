# Skills Overview

This file is included to match the assignment requirement for a clear `skills.md` while keeping the actual Claude-compatible files in Anthropic's `SKILL.md` format.

## Selected Skills

### 1. Codebase Architect

- Folder: `skills/codebase-architect/`
- Purpose: help Claude analyze a repository, identify key modules, map dependencies, and produce a practical engineering plan
- Best for: onboarding, legacy-code analysis, refactor planning, and system design walkthroughs

### 2. MCP Integration Engineer

- Folder: `skills/mcp-integration-engineer/`
- Purpose: guide Claude through designing connectors, mapping tool contracts, and validating secure MCP-based integrations
- Best for: connector planning, internal tooling, external data access, and enterprise workflow automation

### 3. LLM Evals and Guardrails

- Folder: `skills/llm-evals-guardrails/`
- Purpose: structure evaluation plans, test prompts, failure analysis, and safe rollout guidance for LLM features
- Best for: production launch reviews, QA planning, regression checks, and safety-focused iteration

## Why These Three

These were chosen because they align strongly with AI engineering work:

- one skill focuses on understanding codebases
- one skill focuses on tool and connector integration
- one skill focuses on evaluation quality and deployment safety

Together, they form a realistic mini-toolkit for an AI engineer working with Claude.
